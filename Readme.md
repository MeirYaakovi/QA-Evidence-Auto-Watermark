## 📖 Project Overview
This automation tool was developed to solve a common pain point in Manual & Automation testing: The time-consuming task of preparing visual evidence for bug reports.

## 🛠️ Advanced Features
* Batch Media Processing: Efficiently processes dozens of files.
* Intelligent Watermarking: Dynamic overlay of Bug IDs using MoviePy.
* Resolution Adaptation: Automatic handling of Vertical vs. Horizontal orientations.
* Optimized Git Workflow: Professional .gitignore implementation.

## 🚀 Technical Stack
* Language: Python 3.x
* Core Libraries: MoviePy, Pandas.
* Environment: Managed via venv.

## 📂 Project Structure
- process_evidence.py (Automation Engine)
- Evidence/Raw (Input)
- Evidence/Processed (Output)
- .gitignore (Media Exclusion)

## ⚙️ Setup & Execution
1. Clone: git clone https://github.com/meiryaakovi/Netflix.git
2. Activate venv: .\venv\Scripts\activate
3. Install: pip install -r requirements.txt
4. Run: python process_evidence.py
